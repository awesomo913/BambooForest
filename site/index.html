About

I build automation systems grounded in diagnostic analysis for heavy manufacturing machinery — environments where precision and stability aren't optional. That background shapes how I write software: designed for real operating conditions, not clean test environments.

My specialty is Python, custom GUI architecture, and edge computing. I strip out bloat and cloud dependencies to build offline-capable applications that execute reliably on constrained hardware — local AI workflows on micro-hardware, autonomous desktop agents using vision models for keyboard and mouse control, purpose-built memory editors, and JSON-driven audio tooling that owns its own state.

The common thread is direct execution with minimal surface area: software that solves a specific problem completely rather than partially solving several.
